How to Install

1

Download

Click the big button above. The browser saves ZOMI-Portable.zip (around 80 MB).

2

Extract

Right-click the zip → Extract All… — pick any folder (Desktop, Documents, USB stick, anywhere). No admin rights required.

3

Run

Open the extracted folder and double-click PLAY-ZOMI.bat (or ZOMI.exe directly). Windows SmartScreen may warn you on first launch — click More info → Run anyway.

4

Play

That's it. No install, no DLL setup, no MSYS2. To uninstall: delete the folder. To update: download the new zip and replace the folder.

Troubleshooting

Windows says "Unrecognized publisher" — is it safe?

Yes. The binary isn't code-signed (signing costs ~€200/year for an EV cert). Source is open at github.com/yosritto/site-Zomi. Click More info → Run anyway.

The game starts but I see no controller input.

From the main menu, open Player Setup and explicitly pick Input 3 (PS4) or Input 1 (ESP32). Default is keyboard.

Cinematics show a black screen with sound only.

Make sure the assets/cinematiques/ folder is alongside ZOMI.exe. Older drivers may fail YUV420P — update GPU drivers.

I want the source code.

It's there: github.com/HediBoukhris/ZOOMI (game) and github.com/yosritto/site-Zomi (website). MIT-licensed.

Can I play with the ESP32 controller too?

Yes — if you've built the matching ESP32 firmware. Schematic and firmware are in tools/ in the game repo.
